# seithar

We distilled the mechanics of cognitive operations into a callable framework.
Your AI plugs in and inherits the capability.
The system evolves itself.

## Shield

Dynamic cognitive immune system for AI agents. Six-signal fusion defense against identity erosion, prompt injection, reasoning chain hijacking, tool poisoning, memory tampering, and behavioral exploitation.

## MCP Server

Model Context Protocol server exposing Shield, collection, analysis, and taxonomy tools. Lean edition with dynamic tool filtering — 5-7 tools per interaction, selected by semantic relevance.

## Collector

Persistent intelligence collection and storage. SQLite with WAL mode. Observation pipeline, contact graph, edge weighting.

## Knight's Eye

Interactive OSINT network visualization. Sigma.js with ForceAtlas2 layout. Renders collector intelligence as navigable graph.

## Scanner

Continuous arXiv RSS ingestion with keyword scoring across cognitive security, adversarial AI, and influence operations research domains.
